Manage 100 AI interns (agents)

I think of agents as interns operating only with pre-installed tools. Hand them a laptop with Google Calendar, Google Search, and Apollo. Give them context and instructions. They figure out the rest.

They can't install new software—they work within constraints. But those constraints don't dictate every action. They infer next steps from context.

This is my current personal agent setup.

One-off workflows: Perplexity Comet

Example use cases:

  • Finding discounts code at checkout (life saver): it searches through the web and applies every code it could find and gives you the best deal
  • Scraping websites: Extract pricing, product descriptions, feature lists from competitor pages

Key benefits:

  • Zero setup—spin it up instantly
  • Works with login-required tools—log in once, then let it execute
  • No configuration needed for one-time tasks

Recurring workflows: Gumloop with Slack

I built a personal operating system with Slack as the control center. Each channel houses specialized agents with MCP connections to GSuite, Exa (for web search), and Apollo (for contact).

I have 2 channels in my personal Slack (still expanding):

  • Consolidates job postings from 4 Berkeley Slack channels (past 48hrs)
