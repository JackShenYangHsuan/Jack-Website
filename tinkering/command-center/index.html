What if McKinsey consultants work AI-natively

Four years at McKinsey taught me consulting is two jobs: analysis and client counseling.

Analysis

Research, data crunching, synthesis. This consumes most consultant hours. AI automates this if the workflow matches how we think: divide-and-conquer across multiple workstreams, test hypotheses constantly, then consolidate. Control the start (direction) and end (polish & sense-check). Automate the middle.

Counseling

This is where client value lives. Guiding decisions, managing relationships, being in the room when it matters. Right now we don't have enough time for this. AI changes that.

The Shift

AI shifts the balance. Less time on slides, more time with clients. The work goes from analysis-heavy to counsel-heavy. Command Center is my vision to make this shift real.
