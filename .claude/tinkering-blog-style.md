# Tinkering Blog Post Style Guide

## Design Philosophy
- Clean, compact, minimal aesthetic (inspired by Cursor Learn)
- Dense content with tight spacing
- Simple single-column layout
- White background with beige code blocks

## Color Palette
- Background: `#ffffff` (white)
- Text: `#222222` (dark gray)
- Links/subtle text: `#666666` (medium gray)
- Code block background: `#f5f0e8` (beige)
- Code block border: `#e8e0d5` (tan)
- Headings: `#000000` (black)
- Code text: `#333333`

## Typography
- Font family: Inter, system fonts
- Body text: 14px, line-height 1.5
- Intro text: 15px
- H1: 36px, font-weight 400, line-height 1.1
- H2: 22px, font-weight 400
- H3: 16px, font-weight 600
- Code: SF Mono/Monaco, 11px, line-height 1.4
- Back link: 12px

## Spacing
- Body padding: 50px 20px 80px
- H1 margin-bottom: 24px
- H2 margin-top: 36px, margin-bottom: 14px
- H3 margin-top: 24px, margin-bottom: 10px
- Paragraph margin-bottom: 14px
- List margin-bottom: 14px
- List item margin-bottom: 5px
- Code block padding: 14px 16px
- Code block margin: 14px 0

## Layout
- Max width: 700px
- Centered container
- No sidebar
- Simple back link at top

## Mobile Responsive
- Body padding: 40px 16px 60px
- H1: 28px
- H2: 20px
- Body/lists: 13px
- Code: 10px
